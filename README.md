# Pokemon List

This web application showcases a dynamic list of Pok√©mon, seamlessly integrated with the PokeAPI, written in [Next.js](https://nextjs.org/).

## Install dependencies

```bash
yarn install
```

## Running the application

Before running this application, make sure the backend is up and running. You can find the backend app here:
https://github.com/dikiamust/mini-project-be

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
